# 게시물 제목: JAVA FOR문
for문은 변수를 일정 횟수만큼 반복시켜주는 반복문 입니다. 사용 방법은 ``` int b = 0; for(int i=0; i<5; i++){ b++; } ``` 위와 같습니다. 결과는 b가 5가 됩니다. for의 첫째 칸에는 i의 시작 조건 두번째 칸에는 i의 크기 세번째 칸에는 연산 조건을 넣을 수 있습니다.
